Sorted Linked List Project

Overview:

Java application prompts a user to enter integers. Integers will be added
to a linked list, then to be sorted in ascending order and printed in the console.

Prerequisites:

To compile and run the application you will need:

- Java Development Kit (JDK), version 8 or later.

**Compilation is not required for this project as both the .class and .java files are included.

Compilation Instructions:
1. Open a terminal or command prompt.
2. Navigate the 'src' directory of the project.
3. Compile the application using the JDK's 'javac' tool:

"""
javac LinkedListHandler.java SortedLinkedListMain.java
"""

Running the application:

After compilation, run the application from the 'src' directory
with the following command:

java SortedLinkedListMain

Follow the on-screen prompts to enter integers.
When finished inputing numbers, type 'done' and the values will be displayed
